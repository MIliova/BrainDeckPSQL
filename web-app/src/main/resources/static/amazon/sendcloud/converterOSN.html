ℹ️ Information zur Nutzung des Tools ▶
Mit diesem Tool können Sie aus der CSV-Exportdatei von Sendcloud eine Excel-Liste erstellen, die direkt im Browser angezeigt wird. Diese Liste dient als Grundlage für die Erstellung der RE in Sage.

Vorgehensweise:

  1. CSV-Export der erstellten Labels in Sendcloud durchführen.
  2. Falls noch nicht geschehen oder wenn aktualisiert: Die Excel-Datei "Produktkürzel Aktuell" einlesen (0-PfS\Produktdaten\Produktkürzel Aktuell.xlsx).
  3. Auswählen, ob Amazon- oder OSN-Exportlabels eingelesen werden sollen.
  4. Die CSV-Datei(en) auswählen.
  5. Auf Download Excel klicken.
  6. Aus der erzeugten Excel-Datei kann anschließend die RE erstellt werden. Für Amazon-Exporte kann zur Übertragung in Sage auch das Programm ExcelToSage.exe verwendet werden.
